# 🎙️ Text to Speech Converter

A simple and interactive web application that converts written text into speech and speech into text using modern browser APIs.

---

## 🚀 Features

* 🔊 Convert text to speech instantly
* 🎤 Speech to text functionality
* 🌍 Multiple language support
* 🎨 Clean and responsive UI
* ⚡ Fast and lightweight (no backend required)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Web Speech API)

## ▶️ How to Run

### 🔹 Method 1 (Recommended)

1. Open the project in VS Code
2. Install **Live Server extension**
3. Right-click on `text_to_speech.html`
4. Click **"Open with Live Server"**

---

### 🔹 Method 2

* Open `text_to_speech.html` directly in your browser

---

## 💡 How It Works

* Uses the **Web Speech API** for speech synthesis
* Takes user input text and converts it into audio output
* Supports multiple voices and languages
